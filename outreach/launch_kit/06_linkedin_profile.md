## 8. About (2000 char max — usa questa intero)

```
I build infrastructure that makes AI-assisted development safer.

Specifically, I ship DepScope (depscope.dev) — a free, open-source API layer that gives AI coding agents (Claude, Cursor, ChatGPT, Copilot, Aider) live package-health and vulnerability data for 17 ecosystems: npm, PyPI, Cargo, Go, Maven, NuGet, RubyGems, Composer, Pub, Hex, Swift, CocoaPods, CPAN, Hackage, CRAN, Conda, Homebrew.

The problem DepScope solves:
• AI agents recommend packages from training data 6-12 months stale
• They miss recent CVEs (XZ, Log4Shell patterns) and suggest deprecated libs
• They hallucinate package names that never existed
• Each agent hits the registries independently — billions of redundant fetches a day, zero shared cache

DepScope exposes OSV.dev + GitHub Advisory Database as a live API those agents query *before* suggesting an install. One fetch serves every agent. The /api/prompt endpoint returns a token-optimized response ~74% smaller than raw registry JSON.

Stack: FastAPI + PostgreSQL 17 + Redis, MCP server for direct Claude Desktop / Cursor integration, self-hosted in the EU. MIT on GitHub. Operated by Cuttalo srl (IT03242390734).

Background: architect / maker with 15+ years across digital fabrication (CO2 laser cutting, 3D ceramic printing, custom software for bottega work in Grottaglie) and software. I build tools the way I was trained to build things with my hands — solid, unpretentious, fit for purpose.

Always happy to talk about:
• Supply-chain security for AI-assisted codebases
• Building and running infrastructure as a solo founder
• Where the AI coding agent ecosystem is actually headed
• Interesting Italian SaaS / open-source bets
```
